# Project Context

## Project Overview
This project is a React-based web application built with Vite and TypeScript. It appears to be designed for generating "winietki" (vignettes/place cards) or certificates, utilizing PDF generation libraries.

## Tech Stack
- **Languages:** TypeScript, HTML, CSS
- **Frontend:** React 19, Vite
- **PDF Generation:** pdf-lib, pdfjs-dist
- **Utilities:** papaparse (CSV parsing), jszip (Zip creation), file-saver

## Configuration
- **Package Manager:** npm (implied by package-lock.json)
- **Linting:** ESLint (flat config)
- **Formatting:** Prettier (recommended, not explicitly seen but standard)
- **Build:** `npm run build` (tsc + vite build)
- **Dev Server:** `npm run dev`

## Guidelines
- Use TypeScript for all new logic.
- Ensure type safety.
- Prefer functional components and hooks for React.
